# Rock-vs-Mine-Sonar-Prediction

This Machine Learning project uses sonar data to attempt to predict whether an object is a rock or a mine. Data collected from different objects, such as density, material type, and others, are used to model the ability to predict whether the object is a rock or a mine.

# Stack used:
- Python

# Libraries:
- Numpy
- Pandas
- Sklearn

# ML Model:
- Logistic Regression
